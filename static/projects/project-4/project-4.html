This is an extremely simple game I made back in Grade 11. It is a twist on the Google "no internet" dinosaur game where instead of a dinosaur jumping over cacti in the desert, you jump over trash to pick it up in the park. This was my final project for my Grade 11 Computer Science class where we had to make a game using JavaScript that relates to bettering our planet and working towards a greener Earth, which is why I made it so you are picking up trash in the game. You can try the game here: Play A Greener Jump .
